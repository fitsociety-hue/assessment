# Employee Performance Appraisal System

A client-side React application for managing employee performance appraisals, designed for GitHub Pages.

## Features
- **CSV Data Management**: Upload and download evaluation data without a backend.
- **5-Step Evaluation**: Self, Peer, Manager, Staff, and Total Evaluation.
- **Premium UI**: Modern glassmorphism design.
- **PDF Reports**: Automated report generation.

## Getting Started

1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev` to start locally.

## Deployment

This project is configured for GitHub Pages.
1. Build the project: `npm run build`
2. Deploy the `dist` folder.
